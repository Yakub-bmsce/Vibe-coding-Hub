## 🚀 Get Started in 3 Steps

### Step 1: Install (2 minutes)
```bash
npm install
```

### Step 2: Configure (2 minutes)
Create `.env` file:
```env
REACT_APP_GROQ_API_KEY=your_groq_key
REACT_APP_GROW_API_KEY=your_youtube_key
```

### Step 3: Run (1 minute)
```bash
npm start
```

**That's it!** Your app opens at http://localhost:3000

# 🎓 Vibe Learning Hub

> An AI-powered learning platform that makes studying fun, interactive, and personalized.

## 🌐 Live Demo

**👉 [https://vibe-coding-hub-lilac.vercel.app](https://vibe-coding-hub-lilac.vercel.app)**

> Click the link above to open the app. Use the **Demo** button on the login page to try it instantly — no signup needed!!!

---

## ✨ Features

- 🔍 **Smart Search** — Type any topic and get instant domain suggestions
- 📚 **Learn by Concepts** — AI-generated explanations with "Simplify" option
- 🎥 **Learn by Videos** — YouTube tutorials fetched automatically
- 🎮 **Interactive Quizzes** — AI-generated MCQs with scoring and explanations
- 🗺️ **Topic Roadmaps** — Step-by-step learning path for every topic
- 🌍 **Real World Applications** — Industries, jobs, salaries, tools & companies
- 🤖 **AI Chatbot** — Ask anything, get instant help
- 🏆 **XP & Levels** — Gamified learning with streaks and rewards

## 🚀 How to Use

1. Open the [live demo](https://vibe-coding-hub.vercel.app)
2. Click **Demo** to login instantly
3. Search for any topic (e.g., "Python", "React", "Docker")
4. Click a domain → Choose a topic → Learn via 4 tabs!

## 🛠️ Tech Stack

- **Frontend**: React.js, React Router v6
- **AI**: Groq API (llama-3.3-70b-versatile)
- **Videos**: YouTube Data API v3
- **Deployment**: Vercel
- **Theme**: Purple Neon Gradient

## 📦 Local Setup

```bash
git clone https://github.com/Yakub-bmsce/Vibe-coding-Hub.git
cd Vibe-coding-Hub
npm install
```

Create a `.env` file:
```
REACT_APP_GROQ_API_KEY=your_groq_api_key
REACT_APP_GROW_API_KEY=your_youtube_api_key
```

```bash
npm start
```
